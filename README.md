# Talk to AI: A Beginner's Guide to Prompt Engineering

**A course for dummies — no coding experience needed!**

---

## Welcome!

Have you ever typed a question into an AI chatbot and got a weird or unhelpful answer? Or maybe you got an amazing answer and wondered — *how did that work?*

That's what this course is about! You'll learn how to **talk to AI** in a way that gets you the best possible results. This skill is called **Prompt Engineering**, and it's one of the most exciting new skills you can learn today.

---

## What You'll Learn

By the end of this course, you will be able to:
- Explain what Generative AI and Language Models are (in plain English!)
- Understand what a "prompt" is and why it matters
- Use proven techniques (called *patterns*) to write great prompts
- Practice hands-on with Claude, a real AI assistant — for free!

## How to Use This Course

1. Go through the modules **in order** — each one builds on the last
2. Try the **"Your Turn!"** exercises in each module
3. Complete the **Hands-On Lab** in Module 4 using Claude's free version
4. Challenge yourself with the **bonus challenges** at the end!
